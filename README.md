# Constructed Futures Matrix

Three-layer analytical matrix (AI Imaginaries x Construal Level Theory x
Futures Consciousness) with dark and light theme variants. Supporting
figure for an unpublished master's research proposal at Malmö University.

Live site: https://mcmespinaa.github.io/constructed-futures-matrix/

## About this work

This repository contains unpublished academic work developed as part of the
master's programme in Leadership and Organisation for Sustainability (SALSU)
at Malmö University, Sweden. The code and content are shared for academic
demonstration purposes only.

## Citation

If referencing this work, please cite the underlying research proposal:

> Espina, M. C. (2026). *Beyond damage control: How project leaders' mental
> models of human-AI relations shape social sustainability during agentic AI
> transitions* [Unpublished research proposal]. Malmö University, Faculty of
> Culture and Society, Master Programme in Leadership and Organisation for
> Sustainability (OL652E).
